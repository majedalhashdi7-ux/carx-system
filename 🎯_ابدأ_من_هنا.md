## ⚡ الحل السريع (5 دقائق)

### الخطوة 1: افتح Vercel
```
1. اذهب إلى: https://vercel.com/dashboard
2. اختر مشروع: carx-system
3. اضغط على: Settings → Environment Variables
```

### الخطوة 2: احذف المتغير الخاطئ
```
ابحث عن: NEXT_PUBLIC_APIURL
اضغط (⋯) → Delete
```

### الخطوة 3: أضف المتغيرات الصحيحة
```
Name: NEXT_PUBLIC_API_URL
Value: https://hmcar-system.vercel.app/api/v2
Environment: ☑ Production ☑ Preview ☑ Development
[Save]

Name: NEXT_PUBLIC_TENANT
Value: carx
Environment: ☑ Production ☑ Preview ☑ Development
[Save]

Name: NEXT_PUBLIC_APP_NAME
Value: CAR X
Environment: ☑ Production ☑ Preview ☑ Development
[Save]

Name: NEXT_PUBLIC_WHATSAPP
Value: +967781007805
Environment: ☑ Production ☑ Preview ☑ Development
[Save]

Name: NODE_ENV

Environment: ☑ Production
[Save]
```

### الخطوة 4: أعد النشر
```
1. اذهب إلى: Deployments
2. اضغط على آخر deployment
3. اضغط (⋯) → Redeploy
4. انتظر 2-3 دقائق
```
