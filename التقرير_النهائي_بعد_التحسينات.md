## 🚀 كيفية الاستخدام

### 1. تشغيل النظام:
```bash
cd carx-system
npm install
npm run dev
```

### 2. الوصول:
```
http://localhost:3000
```

### 3. الصفحات المتاحة:
- `/` - الصفحة الرئيسية
- `/showroom` - معرض السيارات
- `/showroom/[id]` - تفاصيل السيارة
- `/parts` - قطع الغيار
- `/brands` - الوكالات المعتمدة
- `/login` - تسجيل الدخول
- `/admin` - لوحة التحكم
