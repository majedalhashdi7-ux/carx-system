## 🛠️ الحلول الكاملة

### الحل 1: إعادة تكوين Vercel (الأسهل)

#### الخطوات:
1. اذهب إلى: https://vercel.com/devds-projects/hmcar-client-app
2. انقر: **Settings**
3. تحقق من:
   ```
   Framework Preset: Next.js
   Root Directory: ./
   Build Command: npm run build
   Output Directory: .next
   Install Command: npm install
   Node.js Version: 18.x أو أحدث
   ```
4. انقر: **Save**
5. اذهب إلى: **Deployments**
6. انقر: **Redeploy**

---

### الحل 2: إضافة Environment Variables الناقصة

#### في Vercel Dashboard:
1. اذهب إلى: **Settings** → **Environment Variables**
2. أضف:
   ```
   NEXT_PUBLIC_API_URL=https://hmcar-system.vercel.app/api/v2
   NEXT_PUBLIC_TENANT=hmcar
   NODE_ENV=production
   ```
3. انقر: **Save**
4. **Redeploy**

---

### الحل 3: إصلاح من الكود (الأشمل)

#### نفذ هذه الأوامر:

```powershell
# 1. الانتقال للمجلد
cd C:\hmcar-client-app

# 2. إنشاء .env.production
@"
NEXT_PUBLIC_API_URL=https://hmcar-system.vercel.app/api/v2
NEXT_PUBLIC_TENANT=hmcar
NODE_ENV=production
"@ | Out-File -FilePath .env.production -Encoding utf8

# 3. تحديث vercel.json
@"
{
  "framework": "nextjs",
  "buildCommand": "npm run build",
  "devCommand": "npm run dev",
  "installCommand": "npm install"
}
"@ | Out-File -FilePath vercel.json -Encoding utf8

# 4. رفع التحديثات
git add .
git commit -m "إصلاح: إضافة .env.production و vercel.json"
git push origin main
```
