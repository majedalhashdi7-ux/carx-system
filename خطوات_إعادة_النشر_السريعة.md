# ⚡ خطوات إعادة النشر السريعة

**الهدف:** إصلاح hmcar-client-app.vercel.app

**الوقت المطلوب:** 5 دقائق فقط

---

## 🎯 الخطوات

### 1️⃣ افتح Vercel Dashboard
```
🔗 https://vercel.com/devds-projects/hmcar-client-app
```

### 2️⃣ تحديث Environment Variables

انقر: **Settings** → **Environment Variables**

أضف/حدث هذه المتغيرات:

```
NEXT_PUBLIC_API_URL=https://hmcar-system.vercel.app/api/v2
NEXT_PUBLIC_TENANT=hmcar
NEXT_PUBLIC_APP_NAME=HM CAR
NEXT_PUBLIC_APP_URL=https://hmcar-client-app.vercel.app
NEXT_PUBLIC_WHATSAPP=+967781007805
NODE_ENV=production
```

انقر: **Save**

### 3️⃣ إعادة النشر

انقر: **Deployments**

اختر آخر deployment

انقر: **...** (ثلاث نقاط) → **Redeploy**

انتظر 2-3 دقائق

### 4️⃣ الاختبار

افتح: https://hmcar-client-app.vercel.app

يجب أن تظهر الصفحة الرئيسية ✅
