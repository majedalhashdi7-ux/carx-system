# CAR X System 🚗

نظام CAR X المستقل لمعارض ومزادات السيارات

## 🎯 المميزات

### ✨ التصميم الحديث
- صفحة رئيسية إبداعية مع فيديو خلفية
- بطاقات السيارات بتأثيرات ثلاثية الأبعاد
- بطاقات قطع الغيار المذهلة
- بطاقات الوكالات الدائرية الواضحة

### 🚗 المعرض الكوري
- استيراد السيارات من Encar.com
- صور محسنة ومصلحة
- تفاصيل شاملة للسيارات
- أسعار بعملات متعددة

### 🔧 قطع الغيار
- كتالوج شامل لقطع الغيار
- بحث متقدم بالماركة والموديل
- حالة المخزون المباشرة
- أسعار تنافسية

### 🏢 الوكالات
- عرض دائري للوكالات
- شعارات واضحة ومكبرة
- إحصائيات مفصلة
- ربط مباشر بالمنتجات

## 🚀 التشغيل

### متطلبات النظام
- Node.js 18+
- MongoDB
- npm أو yarn

### التثبيت
```bash
# تثبيت المكتبات
npm install

# نسخ ملف البيئة
cp .env.example .env

# تعديل متغيرات البيئة
nano .env

# تشغيل النظام
npm run dev
```

### البناء للإنتاج
```bash
# بناء المشروع
npm run build

# تشغيل الإنتاج
npm start
```
